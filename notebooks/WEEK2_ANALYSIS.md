# Week 2 - Customer Clustering Analysis

## Objective
Perform customer segmentation using K-Means clustering and analyze churn behavior across segments.

## Method Used
- K-Means Clustering
- Feature Engineering
- Churn Analysis

## Features Used
- tenure
- MonthlyCharges
- TotalCharges

## Segments Identified

### New Customers
- Low tenure
- Higher churn risk

### Premium Customers
- High spending customers
- Lower churn risk

### Regular Customers
- Loyal customers
- Lowest churn rate

## Key Findings
- New Customers showed the highest churn rate.
- Regular Customers showed the lowest churn rate.
- Customer segmentation helps identify retention opportunities.
