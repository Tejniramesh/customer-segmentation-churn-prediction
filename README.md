# 🧠 Customer Segmentation & Churn Prediction using AI

An AI-powered customer intelligence platform that helps businesses analyze customer behavior using Machine Learning, Deep Learning, NLP, and Retrieval-Augmented Generation (RAG).

## 🚀 Features

- 📊 Customer Segmentation using K-Means
- 🛒 Purchase Prediction using LSTM
- 😊 Sentiment Analysis using BERT
- 🤖 AI Customer Insights using RAG + FAISS
- 💻 Interactive Streamlit Dashboard

---

## 🛠 Technologies Used

- Python
- Scikit-learn
- TensorFlow / Keras
- Hugging Face Transformers
- Sentence Transformers
- FAISS
- Streamlit
- Pandas
- NumPy

## ▶️ Run the Project

Clone the repository:

```bash
git clone https://github.com/Tejniramesh/customer-segmentation-churn-prediction.git
```

Install dependencies:

```bash
pip install -r requirements.txt
```

Run the application:

```bash
streamlit run app/app.py
```
